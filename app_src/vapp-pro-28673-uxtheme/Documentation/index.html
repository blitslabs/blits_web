Gulp

Gulp is a task runners that we have used in our template to compile the SCSS.


Gulp Installation

To use Gulp, you need to install it as a global module first throught NPM (if you're new to NPM, you can get to know it better from npm official website):

npm install --global gulp
"sudo npm install --global gulp" for macOS

Now install the dependencies by running (You must have to run this command in the root (source folder of our html package) of your project. This creates node_modules directory with all the plugins in the root of your project.):

npm install

All should be set and you can run the task runner by following command in the root of your project:

gulp watch

Navigation menu option

There are some special classes you can use on you navigation. Please see below.

Example:   <navigation navigation__transparent"></div>
Classes are:
navigation__fluid navigation__fluid is for full width navbar
navigation__border navigation__border will create a border under the nav
navigation__caps navigation__caps will make uppercase the naviagation font
navigation__separate It will help to create a separator/ border between nav item and buttons
navigation__transparent navigation__transparent is for sticky menu
navigation__right The default positioning of the nav is center. So you can change the postion to right by this class.
navigation__left This one is same as avobe but it will change the position to left
navigation-text__dark Use this class if you the hero is white. Basically it makes the nav text dark
navigation-text__gray Use this class if you the hero is white. It makes the nav text gray
navigation__btn-fill The default cta button is outline with transparent background. This class will help you to create a button where button bacground is full.

Buttons

Buttons are divided into types: db-btn__outline, db-btn__type-md, db-btn__type-sm, db-btn__type-icon and other states such as disabled.

Example: <a href="#" class="db-btn db-btn__type-sm">Button</a>
<a href="#" class="db-btn db-btn__type-md">Button</a>
<a href="#" class="db-btn db-btn__outline db-btn__type-sm">Button</a>
<a href="#" class="db-btn db-btn__outline db-btn__type-sm"><i class="db-btn__icon nc-icon nc-apple"></i></a>
db-btn db-btn is the class where all default/ common styles are contained for the buttons. You must have to put this button class for every button.
db-btn__primary db-btn__primary is for default button color that set in the template.
db-btn__yellow, db-btn__orange db-btn__yellow, db-btn__orange etc for different colors for the button. You don't need to use db-btn__primary if use any of these classes..
db-btn__type-sm, db-btn__type-md, db-btn__type-lg Use one of these classes if you want to change the button size.
db-btn__outline Put this class if you want border around the button. Please don't use "db-btn__primary/ db-btn__yellow" if you want border but transparent button.

Changing Theme Color

Changing the theme color is one of the easiest things with sass variable. The 'colors.scss' file is located at "source(sass directory) > assets > src > scss > helpers > variables > _colors.scss". Now all you need is open the file and update the colors with yours.

Variables for colors:    $color-primary: #4777f4;
   $color-white: #ffffff;
   $color-black: #2d3136;
   $color-gray: #818992;
   $color-dark-gray: #2a2f36;
   $color-yellow: #f8e71c;
   $color-blue: #4777f4;
   $color-green: #27ae60;
   $color-violet: #9b59b6;
   $color-red: #e74c3c;

Changing Fonts

You can change the font by re-naming the font name in 'fonts.scss' variable file. The 'fonts.scss' file is located at "source(sass directory) > assets > src > scss > helpers > variables > _font.scss". Now all you need to change the current font name. Such as you want to use Open Sans as primary then change it 'SF Pro Display (Current)' to 'Open Sans'. And place the font css (Such as google font css file) in the html files.

Available Variables for fonts:    $font-primary: 'SF Pro Display';
   $font-secondary: 'Roboto';

SCSS Example:    $font-all: "Open Sans", "Roboto", Helvetica Neue, Arial, sans-serif;
   $font-primary: "Open Sans", "Roboto", Helvetica Neue, Arial, sans-serif;

Markup Example:    <link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet">
